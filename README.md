Fitness Tracker

A modular fitness tracking application developed in Java using JavaFX and Maven.
This project simulates a real-world fitness management system where users can record workouts, track activity, and monitor their progress through a simple graphical interface.

The main purpose of this project is to demonstrate strong understanding of Object-Oriented Programming (OOP), modular architecture, clean code structure, and Java project organization using Maven.

Features:

Record and manage workout sessions

Calculate and track calories burned

Review workout history and activity logs

Monitor personal fitness progress over time

Clean and intuitive JavaFX graphical interface

Well-organized modular project structure

Core logic implemented without external frameworks

Technologies:

Java

JavaFX

Maven
